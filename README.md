# 兰州大学 Touying 幻灯片主题 (Touying Slide Theme for LZU)

本模板基于上海交通大学 Touying 幻灯片主题[touying-sjtu](https://github.com/sjtug/touying-sjtu)修改而来，原主题基于Touying [Dewdrop](https://github.com/touying-typ/touying/blob/main/themes/dewdrop.typ) 主题开发。

![Thumbnail of the project](https://github.com/nftuoa/touying-lzu/blob/main/thumbnail.png)

## 示例

查看 [示例](examples) 即可， 你也可以查看原主题的 [Github Pages](https://sjtug.org/touying-sjtu/) 以获取更多详情。

你也可以自行编译示例文件：

```console
typst compile ./examples/main.typ --root .
```
(用Tinymist可能更好用)

生成的 PDF 文件位于 `./examples/main.pdf`。


## 使用 Github 仓库版本

运行命令 `git clone https://github.com/nftuoa/touying-lzu`，克隆本仓库到本地。

之后你可以参考实例`./examples/main.typ`，基于模板`./template/main.typ`创作自己的幻灯片。
